# Calorie Master 🍱🔥

A prototype Android app for tracking calories and managing food intake.
**WIP (Work In Progress)** — This project is currently under development. Please check back later for updates!

## 🎯 Purpose

This app allows users to:
- Log the foods they eat
- Get information about calorie values
- Track daily intake and progress

## 🛠️ Tech Stack

- Kotlin
- Jetpack Compose
- ViewModel (in progress)
- Material Design

## 🚧 Project Status

Still in early development — currently features basic screen layout and input logic.

## 🔮 Roadmap

- Store data using Room database
- Connect to external APIs for food lookup
- Add visual progress indicators (charts, diagrams)
- Custom goals and user settings
